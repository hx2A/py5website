Description
===========

Disables drawing the stroke (outline). If both ``no_stroke()`` and ``no_fill()`` are called, nothing will be drawn to the screen.

Underlying Java method: `noStroke <https://processing.org/reference/noStroke_.html>`_

Syntax
======

.. code:: python

    no_stroke() -> None
